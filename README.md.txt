\# 🍽️ Zomato Restaurant Analytics Dashboard

\## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze restaurant performance, customer orders, revenue trends, and cuisine popularity using Zomato datasets.

\## 🛠️ Tools Used

* Power BI

* Microsoft Excel

* DAX

\## 📊 Dashboard Features

* Total Orders

* Total Revenue

* Average Order Value

* Average Rating

* Revenue Trend

* Revenue by City

* Revenue by Cuisine

* Payment Mode Analysis

* Interactive Slicers
